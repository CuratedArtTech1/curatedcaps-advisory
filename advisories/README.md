# Security Advisories

This directory contains security advisories for various packages and ecosystems.

## Structure

Advisories are organized by ecosystem:
- `npm/` - Node.js packages
- `pip/` - Python packages
- `maven/` - Java packages
- `nuget/` - .NET packages
- `rubygems/` - Ruby packages
- `go/` - Go packages
- `rust/` - Rust packages

Each advisory follows a standardized format with CVE information, affected versions, and remediation details.
