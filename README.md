# curatedcaps-advisory

A curated database of security advisories for software packages across multiple ecosystems.

## Purpose

This repository serves as a centralized source for tracking and managing security vulnerabilities in software dependencies. It provides structured advisory information to help developers identify and remediate security issues in their projects.

## Structure

- `advisories/` - Security advisory database organized by ecosystem
- Each advisory contains CVE details, affected versions, and remediation guidance

## Contributing

Contributions are welcome! Please ensure all advisories follow the established format and include complete vulnerability information.
